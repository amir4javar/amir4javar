## What I Work On

My work sits at the intersection of **NLP research** and **applied AI engineering**. I'm especially drawn to problems where model efficiency meets real-world utility:

- **Knowledge Distillation** — Investigated multi-teacher vs. single-teacher distillation strategies across GLUE benchmarks. Published a negative result showing that teacher diversity, not quantity, is the key bottleneck. [[repo]](https://github.com/amir4javar/multi_teacher_distillation_experiment)
- **Retrieval-Augmented Generation** — Built a multi-turn RAG system combining hybrid search (vector + BM25), a 4-node LangGraph reasoning pipeline, and real-time streaming over WebSockets. [[repo]](https://github.com/amir4javar/CompArch)
